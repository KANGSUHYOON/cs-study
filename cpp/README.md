# C++ Study

이 디렉토리는 C++을 사용하여 자료구조 및 알고리즘 기초를 학습하고,
문제 풀이 과정에서 헷갈렸던 개념과 사고 흐름을 정리한 공간입니다.

## 구성
- **BOJ**
  - 알고리즘 문제를 유형별로 분류하여 정리
  - 단순 코드 나열이 아니라, 문제 접근 방법과 핵심 개념 위주로 기록
- **Step**
  - 입출력, 조건문, 반복문 등 C++ 문법 및 기초 연습용 문제 정리

## 학습 방식
- 먼저 스스로 로직을 설계하고 구현
- 막힌 부분은 개념을 다시 정리한 후 코드 수정
- 이해한 내용을 Markdown으로 정리하여 재사용 가능하게 기록

## 주요 학습 포인트
- 문자열 처리
- 상태 관리 (visited 배열)
- 조건 분기와 반복 흐름 제어
- C++ bool 조건식과 break의 동작 방식
