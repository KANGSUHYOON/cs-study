# 백준 10820 - 문자열 분석 (C++)

## 📖문제 요약
입력으로 주어지는 여러 줄의 문자열에 대해 각 줄마다 
**소문자 / 대문자 / 숫자 / 공백**의 개수를 구해 출력한다.

## 🫂접근 방법
- 입력이 여러줄이고 공백도 포함. `getline(cin, s)`로 한 줄 전체를 입력받는다.
- 입력이 끝날 때까지(EOF) 반복해야 하므로 `while (getline(cin, s))` 형태로 처리한다.
- 각 줄마다 `cnt[4]` 배열로 소문자/대문자/숫자/공백 개수를 누적한다.

## 💡아이디어
- `cin >> s`는 공백에서 끊기므로 이 문제에는 어울리지 않는다.
- 개수를 세는 배열은 `char`보다 `int`를 사용해야 오버플로우 위험이 적다.
- `ios::sync_with_stdio(false);`와 `cin.tie(nullptr);`는 C++ 입출력 속도 최적화 용도이며,
  `cin/cout`만 사용할 때 안전하다. (`scanf/printf`와 섞어 쓰면 주의)

## ⌨️CODE
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s;

    while (getline(cin, s)) {
        int cnt[4] = {0};

        for (char c : s) {
            if (c >= 'a' && c <= 'z') cnt[0]++;
            else if (c >= 'A' && c <= 'Z') cnt[1]++;
            else if (c >= '0' && c <= '9') cnt[2]++;
            else if (c == ' ') cnt[3]++;
        }

        cout << cnt[0] << ' ' << cnt[1] << ' ' << cnt[2] << ' ' << cnt[3] << "\n";
    }

    return 0;
}
```
## 🤔배운점
 ### 1️⃣코드의 의문
```cpp
while (getline(cin, s)) {
...
}
```
는 
```cpp
while (true) {
  if (getline(cin, s) == false) break;
  ...
}
```
를 한 줄로 줄여쓴 것이다.


### 2️⃣EOF란 무엇일까?
EOF는 **End Of File**의 줄임말로
> **입력이 더 이상 없다**는 신호이다.
> EOF는 더 이상 읽을 줄이 없을 때 발생한다.



### 3️⃣`ios::sync_with_stdio(false);`와 `cin.tie(nullptr);`는 뭘까?
1. `ios::sync_with_stdio(false);`는 C 입출력과 **섞어 쓰지 않고** <br>
동기화를 끊어서 **속도를 빠르게** 하려고 사용한다.
> 기본적으로는 `true`로 되어있다.

<br>
2. 기본적으로 `cin`과 `cout`은 **묶여**있다.
그래서 `cin >> x;`에서는 `cin` 실행 전에 `cout`이 자동으로 출력된다.

#### 문제점
- 입력이 많을 때 매번 `cout`을 비우는 작업이 들어가서 **느려진다.**

#### 해결 방법
```cpp
cin.tie(nullptr);
```
➡️ `cin`이 실행될 때 `cout`을 신경 안 쓰게 한다.
