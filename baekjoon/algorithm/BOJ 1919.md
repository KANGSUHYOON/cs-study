# 백준 1919 - 애너그램 만들기 (c++)

## 📚문제 요약
두 단어가 주어질 때, 두 단어를 애너그램으로 만들기 위해 제거해야 하는 문자 개수의 합을 구하라.

## 💡아이디어
- 문자 개수의 합을 구하라 하니 인덱스 `c - 'a'`를 이용하여 개수를 구하자.
- 알파벳은 26개이니 길이 26개짜리 배열로 각 문자의 개수를 세자.
- 첫 번째 문자열은 `+1`, 두 번째 문자열은 `-1`로 처리하면
  같은 문자는 서로 상쇄되고 남은 개수만 배열에 남게 될 것이다.
- `#include <cmath>`를 이용하여 `abs`(절댓값)을 구하면 총 합이 나올 것.

## ❗주의점
- 배열 초기화는 꼭 해주자. 쓰레기값이 생겨서 오류난다.

## ⌨️CODE
```cpp
#include <iostream>
#include <string>
#include <cmath>   
using namespace std;

int main() {
    string a, b;
    cin >> a >> b;
    int alpha[26] = {0};

    for (char c : a) {
        alpha[c - 'a']++;
    }
    for (char c : b) {
        alpha[c - 'a']--;
    }

    int result = 0;
    for (int i = 0; i < 26; i++) {
        result += abs(alpha[i]);
    }

    cout << result << "\n";
    return 0;
}

```
## 👨‍🎓배운점
- 이제야 인덱스 c - 'a'를 언제 사용할 지를 깨닳았다.
- `for (int i = 0; i < a.length(); i++)`은 `for (char c : a)`와 완전히 같은 의미이다.
