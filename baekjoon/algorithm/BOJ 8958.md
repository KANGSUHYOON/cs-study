# 백준 8958 - OX 퀴즈 (C++)

## 📖문제 요약
문자열을 입력하고 O가  **연속으로 나오면 +1을 계속 누적**, X가 나오면 누적된 O의 값을 **초기화** 시킨다.
ex) OOOXXXOXOOXXX -> +1 +2 +3 0 0 0 +1 0 +1 +2 0 0 0 = 10
마지막 값을 출력하라.

## 🫂접근 방법
- 'O'가 연속으로 나오면 +1을 누적한 값을 계속 더해야 하므로 연속된 값을 저장하는 변수와 점수 변수를 따로 만들자
- 'X'가 나오는 순간 누적된 O의 값을 초기화 시키자

## ⌨️CODE
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int TC;
    cin >> TC;

    while (TC--) {
        string S;
        cin >> S;

        int score = 0;
        int count = 0;

        for (char ch : S) {
            if (ch == 'O') {
                count++;
                score += count;
            } else {
                count = 0;
            }
        }

        cout << score << "\n";
    }
    return 0;
}
```

## 배운점 / 느낀점
- 이 문제는 연속된 상태를 변수로 유지하며 누적 점수를 계산하는 문제였다.
- 나 혼자 생각을 하고 힌트와 도움 없이 문제를 푸는 연습이 되었다.
