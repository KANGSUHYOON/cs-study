# 백준 2908 상수

## 문제 요약
**수를 거꾸로 읽는 상수**에게 상근이가 수의 크기 비교 문제를 내줬다. <br>
상근이는 **두 수 A, B(세자리 수 이고 0이 포함되어 있지 않다.)** 를 주고 그 중 큰 수를 말해보라 했다.<br>
두 수가 주어졌을 때, 상수의 대답을 출력하는 프로그램을 작성하여라.

## ⌨️입력
A, B를 입력한다.

## 🖨️출력
첫째 줄에 상수의 대답을 출력한다.

## 💡아이디어
- 우선 숫자를 뒤집으려면 `for`문을 이용하여 입력받은 숫자를 거꾸로 출력 시키면 된다.
- 거꾸로 된 숫자를 저장시킬 rev1, rev2를 선언한다.
- 수를 문자열로 받아야하기 때문에 `int`가 아닌 `string`으로 받아야한다.
- 문자열을 누적하기 위해 `+=` 을 사용하자

## 📌완성 코드
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
  string A, B;
  cin >> A >> B;
  string rev1 = "";
  string rev2 = "";

  for (int i = 2; i >= 0; i--) {
    rev1 += A[i];
    rev2 += B[i];
  }
  if (rev1 > rev2) {
    cout << rev1;
  }
  else {
    cout << rev2;
  }
  return 0;
}
```
## ⚠️실수했던 부분 및 배운 점
- `rev1 = A[i]`가 아니라 `rev1 += A[i]`로 해야 문자열이 누적된다.
- 문자열을 누적할 때는 누적시킬 변수에 `""`로 선언해주자.
  > 가독성이 좋아진다. `= ""`를 붙이면 "이 변수는 누적용이구나"를 빠르게 인지 가능함.
