# 백준 1152번 단어 개수

## 문제
- 영어 대소문자와 공백(' ')으로 이루어진 문자열 주어짐. 이 문자열에는 몇개의 단어가 있는가?
이를 구하는 프로그램을 작성하라.
단, 한 단어가 여러번 등장하면 등장 횟수만큼 모두 카운트 한다.

## 입력
- 첫 줄에 영어 대소문자와 공백으로 이루어진 문자열 주어짐.
문자열 길이는 1,000,000을 넘지 않는다. 
단어는 공백 1개로 구분되고 공백은 연속되지 않는다.
문자열 시작과 끝에 공백 허용

## 출력
- 첫째줄에 단어의 개수 출력

## 나의 아이디어
> 1. 문자열 입력 -> 공백으로 단어구분(반복문으로 " " 찾기) -> 단어 개수만큼 카운트
> 2. 단어가 4개일 시 공백의 경우의 수
> > ' 'word' 'word' 'word' 'word' '<br>
> > ' 'word' 'word' 'word' 'word <br>
> > word' 'word' 'word' 'word' ' <br>
> > word' 'word' 'word' 'word
> 3. 공백의 앞, 뒤로 단어가 있을 경우 하나로 카운트 하면 되나?   

## 헷갈린 포인트
- 공백 개수로 단어 수를 유추하려 하면 안된다.
- 시작 공백, 끝 공백이 있는 경우 예외가 생김.
- " "처럼 공백만 있는 문자열도 입력될 수 있음.
=> 공백을 세는 접근은 불안정하다.

## 다시 생각해본 아이디어
" "이 아닌 "단어가 시작되는 순간"을 세자.
* 현재의 문자가 공백이 아님 + 문자열의 맨 앞이거나 바로 앞의 문자가 공백 = 단어가 시작되는 순간
> 이를 조건식으로 표현하면
> ```cpp
> S[i] != ' ' && (i == 0 || S[i - 1] == ' ')
> ```

## 최종 코드
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
  string S;
  getline(cin, S);

  int count = 0;

  for (int i = 0; i < (int)S.size(); i++) {
    if (S[i] != ' ' && (i ==0 || S[i - 1] == ' ')) {
      count++;
    }
  }
  cout << count <<endl;
  return 0;
}
```

## 배운 점
- 문자열 문제는 상태 변화를 보는게 안전하다.
- getline() 은 공백 포함 문자열 입력에 무조건 쓰인다.
- 조건식 &&(AND), ||(OR)를 유연하게 쓸 수 있도록 하자.
