# 백준 5222 - 다이얼 (C++) 

## 📖문제 요약
전화 다이얼의 알파벳을 입력받아, 각 문자를 누르는 데 걸리는 시간을 모두 합해 출력한다.

## 🫂접근 방법
- 입력 문자열을 한 글자씩 순회한다.
- 각 문자 `c`가 어느 알파벳 범위에 속하는지 `if / else if`로 판단한다.
- 범위에 맞는 시간을 `sec`에 누적한다.

## 💡아이디어
알파벳 구간별로 시간이 고정되어 있으므로,
`'A' ~ 'C'`, `'D' ~ 'F'` 처럼 **문자 범위 비교**로 시간을 매핑한다.

## ⌨️CODE
```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    string s;
    cin >> s;
    int sec = 0;

    for (char c : s) {
        if (c >= 'A' && c <= 'C') sec += 3;
        else if (c >= 'D' && c <= 'F') sec += 4;
        else if (c >= 'G' && c <= 'I') sec += 5;
        else if (c >= 'J' && c <= 'L') sec += 6;
        else if (c >= 'M' && c <= 'O') sec += 7;
        else if (c >= 'P' && c <= 'S') sec += 8;
        else if (c >= 'T' && c <= 'V') sec += 9;
        else if (c >= 'W' && c <= 'Z') sec += 10;
    }

    cout << sec;
    return 0;
}
```

## 🔖나의 실수 / 배운 점
- `sec`는 이미 선언된 변수인데, 조건문 안에서 `int sec += 3;` 처럼
  `int`를 다시 붙이면 **재선언(컴파일 에러)**이 발생한다.
- C++에서 `char`는 `'A' <= c && c <= 'C'` 처럼 **문자끼리 범위 비교**가 가능하다.
