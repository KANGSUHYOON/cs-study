# BOJ 2577 숫자의 개수(c++)

## 📚문제 요약
세 정수 `A, B, C`를 입력받고 <br>
`A × B × C`의 결과에 포함된 숫자 `0 ~ 9`가 <br>
각각 몇 번 등장하는지 출력하는 문제.

## 💡핵심 아이디어
배열을 **저장용이 아니라 카운팅 용도**로 써야한다.
```cpp
int Cnt[10] = {0};
```
- `Cnt[0]` → 숫자 0 등장 횟수
- `Cnt[1]` → 숫자 1 등장 횟수
- ...
- `Cnt[9]` → 숫자 9 등장 횟수

## 🫂풀이 방법
### 숫자 하나씩 추출하기
```cpp
while (mul > 0) {
  int digit = mul % 10;
  Cnt[digit]++;
  mul /= 10;
}
```
#### ❓왜 `%10`을 사용할까?
- `12345680 % 10 = 0`
- 마지막 자릿수를 얻을 수 있다

#### ❓왜 `/= 10`을 사용할까?
- `12345680 / 10 = 1234568`
- 숫자를 한 자리씩 줄여서 전체 탐색이 가능해진다

## 😞나의 실수
원래는 문자열로 변환해서 풀 계획이었다.
```cpp
string mulStr = to_string(mul);
```

다만,
### 📏실수 1. 문자열 길이를 고려하지 않았다.
```cpp
for (int i = 0; i < 10; i++) {
  Cnt[mulStr[i]]++;
}
```
#### 문제점
곱한 결과가 항상 10자리가 아니다.<br>
이는 존재하지 않는 index에 접근하게 되어 runtime error가 발생한다.
이는 
```cpp
for (int i = 0; i < mulStr.length(); i++)
```
로 해결이 가능하다

### 🫵실수 2. 문자 자체를 index로 사용하였다.
```cpp
Cnt[mulStr[i]]++;
```
여기서 `mulStr[i]`는 문자다.
즉 문자 내부적으로 `'7'` 이 들어가게 된다면,<br>
`Cnt['7']++;` → `Cnt[55]++;`가 되어 범위를 초과하게 된다. <br>
이는 아스키 코드에서 문자를 숫자로 바꾸는 `Cnt[mulStr[i] - '0']++;`로 바꾸면 해결된다.

## ⌨️최종 CODE
```cpp
#include <iostream>
using namespace std;

int main() {
    int A, B, C;
    cin >> A >> B >> C;

    int mul = A * B * C;

    int Cnt[10] = {0};

    if (mul == 0) {
        Cnt[0] = 1;
    } else {
        while (mul > 0) {
            int digit = mul % 10;
            Cnt[digit]++;
            mul /= 10;
        }
    }

    for (int i = 0; i < 10; i++) {
        cout << Cnt[i] << '\n';
    }

    return 0;
}
```
## 😃배운 점

### 🔥배열은 단순 저장뿐 아니라 "빈도표"로 활용 가능하다
```cpp
Cnt[digit]++;
```
### 🔥문자 숫자를 index로 쓰려면 변환이 필수!
```cpp
digit = mulStr[i] - '0'
```

## ➕다른 방법으로도 풀어보자.(CODE)
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
	int A, B, C;
	cin >> A >> B >> C;
	int mul = A * B * C;

	string mulStr = to_string(mul);
	int Cnt[10] = { 0 };
	for (int i = 0; i < mulStr.length(); i++) {
		Cnt[mulStr[i] - '0']++;
	}
	for (int i = 0; i < 10; i++) {
		cout << Cnt[i] << endl;
	}
	return 0;
}
```
