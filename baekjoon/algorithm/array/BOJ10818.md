# 배열 알고리즘 문제 정리 (BOJ 10818 최소, 최대)

## 문제 정리
- 정수 N개가 주어졌을 때 최솟값과 최댓값을 구하는 문제
- 정렬을 사용하지 않고 직접 비교를 통해 해결

## 접근 방법
1. 입력된 값을 배열에 저장
2. 첫 번째 값을 기준으로 최소값과 최대값 초기화
3. 배열을 순회하며 최소값, 최대값 갱신

## CODE
```cpp
#include <iostream>
using namespace std;

int main() {
	int N;
	cin >> N;

	int arr[1000000];

	for (int i = 0; i < N; i++) {
		cin >> arr[i];
	}

	int min = arr[0];
	int max = arr[0];

	for (int i = 1; i < N; i++) {
		if (arr[i] < min) min = arr[i];

		if (arr[i] > max) max = arr[i];

	}
	cout << min << " " << max << endl;
	return 0;
}
```
## 포인트
- 초기값을 임의의 값이 아닌 `arr[0]`으로 설정
- 최소값과 최대값 비교는 서로 독립적인 조건문으로 처리

## 배운 점
- 정렬(`sort`)은 편리하지만, 최소/최대 탐색 문제에서는 불필요한 연산이 될 수 있음
> 알고리즘적으로 위의 코드는 O(N)의 시간복잡도를 갖지만, `sort`는 O(N log N)의 시간복잡도를 갖는다.
