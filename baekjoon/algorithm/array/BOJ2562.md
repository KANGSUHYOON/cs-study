# BOJ 2562(Maximum)

## 문제 핵심
- 서로 다른 자연수 9개가 주어질 때
- 최댓값과 그 값이 몇 번째 입력인지 출력하는 문제

## 접근 방법
1. 배열을 사용하여 입력값을 순회하자
2. 첫 번째 입력값을 기준으로 최댓값을 초기화 하자
3. 이후 입력값과 비교하며 더 큰 값이 나오면
    - 최댓값 갱신
    - 해당 위치(인덱스) 함께 갱신

## 핵심
- 최댓값 초기화를 임의의 값이 아닌 **첫 입력값 기준**으로 설정
- 값이 갱신될 때 위치 정보도 함께 관리
- 정렬 없이 한 번의 반복문으로 해결

## CODE
```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[9];
    int maxVal;
    int index = 0;

    for (int i = 0; i < 9; i++) {
        cin >> arr[i];

        if (i == 0 || arr[i] > maxVal) {
            maxVal = arr[i];
            index = i + 1; // 문제는 1부터 센다
        }
    }

    cout << maxVal << "\n" << index << endl;
    return 0;
}
```
## 배운 점
- **값과 관련된 정보(위치)**를 함께 관리해야 한다.
- `i == 0`조건을 사용하면 안전하게 기준값을 초기화할 수 있다.
