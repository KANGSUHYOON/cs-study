# 백준 알고리즘 알파벳 찾기(10809번)

## 문제
> 알파벳 소문자로만 이루어진 단어 S가 주어진다. 각각의 알파벳에 대해서, 단어에 포함되어 있는 경우에는 처음 등장하는 위치를, 포함되어 있지 않은 경우에는 -1을 출력하는 프로그램을 작성하시오.

## 입력
> 첫째 줄에 단어 S가 주어진다. 단어의 길이는 100을 넘지 않으며, 알파벳 소문자로만 이루어져 있다.

## 출력
> 각각의 알파벳에 대해서, a가 처음 등장하는 위치, b가 처음 등장한느 위치, ...z가 처음 등장하는 위치를 공백으로 구분해서 출력한다.
> 만약, 어떤 알파벳이 단어에 포함되어 있지 않다면 -1을 출력한다. 단어의 첫 번째 글자는 0번째 위치이고, 두번째 글자는 1번째 위치이다.

### 내가 생각해본 것
> 1. "일단 string S; 나 char S[100]; 을 정의하고 시작하자."
> 2. "출력할 때 위치를 공백으로 구분하니 " "를 사용해야겠구나."
> 3. "알파벳 배열을 따로 만들어서 반복문을 사용해야겠다."
> 4. "알파벳 배열 int alpha[26]을 사용하자."
> 5. "알파벳 배열 26개를 모두 -1로 초기화하고 반복문을 통해 값을 넣자"

## 완성 코드
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
  string S;
  cin >> S;

  int alpha[26];  

  for (int i = 0; i < 26; i++) {
    int idx = S[i] - 'a';
    if (alpha[idx] == -1) {
      alpha[idx] = i;
    }
  }
  for (int i = 0; i < 26; i++) {
    cout << alpha[i] << " ";
  }
  return 0;
}
```
