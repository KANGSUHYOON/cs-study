# 문자열 string과 char에 대한 공부
***

### 백준 10926번
> 준하는 사이트에 회원가입을 하다가 joonas라는 아이디가 이미 존재하는 것을 보고 놀랐다.
> 준하는 놀람을 ??!로 표현한다.
> 준하가 가입하려고 하는 사이트에 이미 존재하는 아이디가 주어졌을 때, 놀람을 표현하는 프로그램을 작성하시오.

### 입력
> 첫째 줄에 준하가 가입하려고 하는 사이트에 이미 존재하는 아이디가 주어진다.
> 아이디는 알파벳 소문자로만 이루어져 있으며, 길이는 50자를 넘지 않는다.

### 출력
> 첫째 줄에 준하의 놀람을 출력한다. 놀람은 아이디 뒤에 ??!를 붙여서 나타낸다.

## 나의 접근 방식
- "문자열을 50자로 제한하였다. char[51]을 쓰면 되겠구나"
- "단순하게 입력을 하고 뒤에 ??!만 붙이면 되겠구나"

## 코드
```cpp
#include <iostream>
using namespace std;

int main() {
  char str[51];
  cin >> str;
  cout << str << "??!";
  return 0;
}
```
> 결론적으로 보면 맞는 코드이고 두번 틀렸다. 처음에는 잘못적은걸 그대로 냈다가 컴파일 오류가 났고, 두번째로 낸것은 return 0;을 {} 밖으로 빼서 적었다.
> 실수라고는 안하겠다. 내가 검토를 못한거지 뭐...

## 느낀점
- 생각해보면 그냥 string a;로 정의를 하고 해도 되겠다 싶었다.
> ```cpp
> include <iostream>
> using namespace std;
>
> int main() {
>   string a;
>   cin >> a;
>   cout << a << "??!" << endl;
>   return 0;
> }
> ```
> > 이렇게 해도 정답이였다.

## 배운점
- int main() 이 끝나면 cpp에서는 자동으로 return 0;이 붙는다.
- 삼중자 ??! -> | 로 변하는것을 해결하려면 ?\?!로 역슬래시를 넣으면 해결된다.
